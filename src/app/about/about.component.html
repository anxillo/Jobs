Jobs Scraper Beta

Three component jobs scraper from: www.job-room.ch

www.job-room.ch has is search results limited to 100 records, hence the need to scrap the results in a separated database to have all results displayed.

due to this limitation is also not possible to store all the job records, so the web app only stores 3 different types: all jobs in Ticino, all jobs with the keywords "web", "online" and "digital". The different types are correctly labeled in the results page.

GitHub repositories:

Python scraper and SQLite management: https://github.com/anxillo/ScrapJobs

RESTful PHP API for SQLite (third party): https://github.com/alixaxel/ArrestDB/

AngularJS frontend: https://github.com/anxillo/Jobs
